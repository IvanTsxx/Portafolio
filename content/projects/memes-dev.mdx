## Idea

MemesDev es una plataforma open-source donde desarrolladores pueden **compartir y descubrir memes de programación** en un entorno moderno, rápido y fluido.

El objetivo es combinar entretenimiento con una **implementación técnica seria**, mostrando cómo construir una aplicación real con buenas prácticas.

## Características

- Autenticación con **Magic Link** y proveedores sociales
- Interfaz rápida, **minimalista** y accesible
- Experiencia centrada en la **comunidad**
- Flujo de interacción simple para publicar y explorar contenido

## Enfoque técnico

El proyecto fue diseñado como una **aplicación de producción**, no como un experimento descartable.

Principios principales:

- Arquitectura **clara y mantenible**
- Tipado estricto **end-to-end**
- Validación consistente con **Zod**
- Sistema de autenticación **bien integrado**
- Separación de responsabilidades **en capas**

## Resultado

Un proyecto open-source que combina humor con **buenas prácticas de ingeniería**, útil tanto como plataforma funcional como referencia técnica para otros desarrolladores.
